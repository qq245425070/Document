### 占位符
%d	整数  
%f	浮点数  
%s	字符串  
%x	十六进制整数  
```
print('%d--%s' % (10, 'alex'))  
```
### format  
```
print('Hello, {0}, 成绩提升了 {1:.1f}%'.format('小明', 17.125))
```
