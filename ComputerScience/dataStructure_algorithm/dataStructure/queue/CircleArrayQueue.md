###### 循环队列

逻辑上首尾相连的顺序存储结构称为循环队列。

> 队空 与 队满

保留一个元素的存储空间。此时，队列满时的判断条件为  (rear + 1) % maxSize == front；队列空的判断条件还是front == rear
```
__, __, __, __, __
00, __, __, __, __
00, 11, __, __, __
00, 11, 22, __, __
00, 11, 22, 33, __
00, 11, 22, 33, 44
__, 11, 22, 33, 44
__, __, 22, 33, 44
AA, __, 22, 33, 44
AA, __, 22, 33, 44

```
